# RecoverPlan

## Adaptive Study Scheduling via Predictive Modeling in C++

RecoverPlan is an academic C++ project that models the problem of study planning as a predictive scheduling problem.  
The system estimates the expected completion time of study tasks using Linear Regression, predicts the probability of delay using Logistic Regression, and generates an adaptive study schedule based on task priority, available time, and recovery strategies.

## Project Objective

The objective of this project is to design and implement an object-oriented C++ framework that helps students organize their study tasks more effectively by combining scheduling logic with basic machine learning models.

## Problem Statement

Students often underestimate the time required to complete academic tasks, which leads to delayed submissions, missed study sessions, and poor time management.  
RecoverPlan addresses this issue by predicting task duration and delay risk, then automatically adjusting the study plan when delays occur.

## Methodology

The project uses:

- Object-Oriented Programming in C++
- Linear Regression for task duration prediction
- Logistic Regression for delay probability estimation
- Priority-based scheduling
- Recovery planning for missed or delayed tasks

## System Components

1. Task Management
2. Student Profile Management
3. Study Session Management
4. Time Prediction Module
5. Delay Prediction Module
6. Scheduling Engine
7. Recovery Engine
8. Report Generation
